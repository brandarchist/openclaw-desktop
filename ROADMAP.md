# Roadmap

The expected progression is intentionally simple:

1. repository skeleton and directional documentation
2. minimal desktop shell using Electron
3. basic chat-oriented application surface
4. runtime seam introduction
5. startup and preflight checks
6. inspector and debug surface
7. initial OpenClaw integration
8. packaging proof-of-concept for desktop distribution

Notes:

- Early phases are about structure and attachment points more than functionality.
- Lightweight local model support is a future exploration, not part of the early phases above.
