# openclaw-desktop

`openclaw-desktop` is an intentionally early repository skeleton for a local-first OpenClaw desktop/appliance direction.

It exists to make the intended shape of the project legible: a desktop shell, a local runtime seam, future OpenClaw attachment points, and the kinds of inspection and preflight surfaces a serious local artifact will likely need.

It is not a production system, a secure appliance, a complete desktop app, or an OpenClaw integration. The current state is deliberately skeletal.
